# 实习汇报系统 - 前端

## 技术栈
- Vue 3 + TypeScript + Vite
- Element Plus + Pinia + Vue Router
- Axios + ECharts

## 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器（默认端口 5173）
npm run dev

# 构建生产环境
npm run build
```

## 开发提示
- 启动后会输出 `Local` 和 `Network` 地址
- 如需局域网访问，请确认 `package.json` 中 `dev` 脚本包含 `--host` 参数
- 后端 API 默认地址：`http://localhost:5000`
